# 音乐小课堂 - 小学生学音乐 Web 应用

一个专为小学生设计的互动音乐学习应用，让学习音乐变得简单有趣！

## 🎵 功能特点

### 1. 📚 音乐知识学习
- **认识音符**：学习基本的 7 个音符（Do、Re、Mi、Fa、Sol、La、Si）
- **音符时长**：了解全音符、二分音符、四分音符等不同时长的音符
- **认识谱号**：学习高音谱号和低音谱号

### 2. 🎯 识谱练习
- 随机播放音符，让小学生练习听音识别
- 实时计分系统，答对加分，答错扣分
- 视觉反馈，正确/错误都有明显的动画提示

### 3. 🥁 节奏练习
- 可调节速度（40-160 BPM）
- 支持多种节拍（4/4 拍、3/4 拍、2/4 拍）
- 视觉节拍显示，跟着节拍练习
- 音频节拍器辅助练习

### 4. 🎹 虚拟钢琴
- 8 个白键和 5 个黑键
- 支持鼠标点击演奏
- 支持键盘快捷键（A-K 键）
- 真实的钢琴音色

## 🚀 如何使用

### 方法一：直接打开
1. 下载所有文件到同一文件夹
2. 双击 `index.html` 文件
3. 在浏览器中打开即可使用

### 方法二：使用本地服务器
```bash
# 使用 Python 3
python -m http.server 8000

# 使用 Node.js http-server
npx http-server

# 使用 PHP
php -S localhost:8000
```
然后在浏览器中访问 `http://localhost:8000`

## 🎨 技术栈

- **HTML5**：页面结构
- **CSS3**：现代化响应式设计
- **JavaScript (ES6+)**：应用逻辑
- **Web Audio API**：音频生成和播放

## 💡 使用说明

### 学习知识
- 点击不同的知识卡片查看详细内容
- 每个音符都有播放按钮，可以听到对应的声音
- 点击右上角的 ✕ 关闭详细内容

### 识谱练习
1. 点击"播放音符"按钮
2. 仔细听播放的音符
3. 选择你认为正确的音符
4. 系统会自动判断并计分

### 节奏练习
1. 调节速度滑块设置 BPM
2. 选择节奏型（4/4、3/4、2/4）
3. 点击"开始"按钮
4. 跟着节拍拍手或点头
5. 按"暂停"按钮停止

### 虚拟钢琴
- **鼠标点击**：直接点击琴键演奏
- **键盘操作**：
  - A → Do (C4)
  - S → Re (D4)
  - D → Mi (E4)
  - F → Fa (F4)
  - G → Sol (G4)
  - H → La (A4)
  - J → Si (B4)
  - K → 高音 Do (C5)
